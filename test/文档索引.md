## 💡 使用提示

### ⚠️ 重要提醒

1. **客户名称必须完全一致**
   - 生成时使用什么名称，验证时就必须用什么名称
   - 包括大小写、空格、标点符号都要一致

2. **保护私钥文件**
   - `new_private.pem` 是签名许可证的关键
   - 不要提交到代码仓库
   - 不要部署到生产环境

3. **使用正确的导入路径**
   - 必须使用 `/v2` 路径
   - 例如：`github.com/unidoc/unioffice/v2/document`

4. **配置 go.mod**
   - 需要正确的 `replace` 指令
   - 指向本地修改的 unioffice 代码

## 🔧 常用命令

```bash
# 生成许可证
go run license_gen.go -customer "名称" -output license.key -privkey new_private.pem

# 调试模式
go run license_gen.go -customer "名称" -debug -privkey new_private.pem

# 运行测试
go run test_license.go    # 基础测试
go run test_full.go       # 完整测试

# 查看许可证内容
cat license.key | sed -n '2p' | base64 -d | jq .

# 清除缓存
go clean -cache -modcache
```
